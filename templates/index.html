Key Project Highlights

  • Logistic Regression, Navie Bayes, Random Forest, and XG Boost are used for model training. XG Boost is the best performing in terms of F1 Score. XG Boost is run for parameter tuning and calibrated with best parameters for final model building. As obvious, XG Boost is used for this web application deployment.
  • Feature engineering is performed and generated word count, verb count, noun count and number counts.
  • Stacking method is used for X data input, which is merging with actual cleaned text and data points from feature engineering

Future Improvement

  1. Dataset is imbalanced and need to treat with SMOTE or other sampling method.
  2. Try a few variations of data input: e.g. stand-alone cleaned text
  3. Use Kfold or other cross validation to make model generalized learning.
